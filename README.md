# EAGLAUNCHER

A launcher created to improve your gameplay experience on EaglerCraft.

---

## Project Overview

EAGLAUNCHER is a web-based launcher that allows users to access different EaglerCraft versions.

It includes official versions as well as modded clients. This project was heavily inspired by irv77.
