📖 ABOUT EAGLERCRAFT

What is Eaglercraft?

Eaglercraft is an open source Minecraft that you can play in any regular web browser. That includes school chromebooks. You can join real Minecraft 1.5.2 servers with it through a custom proxy plugin developed for Bungeecord and Velocity.

How does it work?

Eaglercraft uses the launcher assets of the official version of Minecraft direct from Mojang to compile it to Javascript using TeaVM. A custom compatibility layer by LAX1DUDE allows the OpenGL 1.3 rendering engine to operate through an HTML5 WebGL canvas.

Is this legal?

Yes. All assets protected by DMCA are not hosted by Eaglercraft — they are acquired directly from Mojang's public servers, exactly like the original Minecraft Launcher does.

Who made it?

LAX1DUDE created Eaglercraft in late 2021 (first stable build February 2022). Ayunami2000 contributed heavily. PeytonPlayz595, EymenWSMC, and radmanplays ported other Minecraft versions.
